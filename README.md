## 📖 Project Overview
This project implements a Simple Linear Regression model entirely from scratch without using any high-level machine learning libraries like `scikit-learn`. The core logic is built using raw mathematics, `NumPy`, and `Pandas` to understand how the algorithm works under the hood.

## ✨ Features
* **Built from Scratch:** Calculates Mean Squared Error (MSE) and uses **Gradient Descent** to iteratively find the optimal slope ($m$) and intercept ($b$) for the line of best fit.
* **Interactive Prediction:** Allows users to input 'Study Hours' in the terminal to get predicted 'Marks'.
* **Data Visualization:** Uses `Matplotlib` to plot the actual data points and the generated Best Fit Line.

## 🛠️ Technologies Used
* Python
* NumPy
* Pandas
* Matplotlib
